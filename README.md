# Kanban Board

A responsive Kanban Board application that helps users organize and manage tasks using a drag-and-drop interface.

## Features

- Create new tasks
- Edit existing tasks
- Delete tasks
- Drag and drop tasks between columns
- Responsive design for desktop and mobile
- Clean and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
